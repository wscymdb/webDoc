# HTML元素

## 1.html的属性

### 1.1. ``lang``

#### 1.1.1. 作用

* 帮助``语音合成工具``确定使用的发音

* 帮助``翻译工具``确定使用的翻译规则

#### 1.1.2. 常用的规则 

* lang="en"  表示这个HTML文档的语言是英文的
* lang="zh-CN"  表示这个HTML文档的语言是中文（简体）的

```html
<html lang="zh-CN">
    ...
</html>
```

# URL和URI

URL(Uniform Resource Locator): 统一资源定位符，俗称``网络地址``，相当于``网络中的门牌号``

URI(Uniform Resource Identifier): 统一资源标识符，用于标识``Web技术使用的逻辑或物理资源``

URL作为一个网络web资源的地址，可以唯一将一个资源识别出来，所以URL是一个URI

URL是URI的一个子集

但是URI不一定是URL
